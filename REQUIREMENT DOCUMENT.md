### Business Requirement

#### User should be able to:

1\. Add products to cart  
2\. Modify cart items  
3\. Apply coupon codes  
4\. Select delivery address  
5\. Choose payment method  
6\. Place order successfully  
7\. Receive order confirmation

##### Functional Requirements

FR\_01: User can add a product to the cart    
FR\_02: Cart should display product details (name, price, quantity)    
FR\_03: User can update quantity    
FR\_04: The user can remove the item    
FR\_05: System calculates subtotal, tax & total correctly    
FR\_06: The user can apply a valid coupon    
FR\_07: Invalid coupon shows an error    
FR\_08: User can select saved address    
FR\_09: User can add a new address    
FR\_10: The user can select the payment method    
FR\_11: Payment success redirects to the confirmation page    
FR\_12: Unique Order ID is generated    
FR\_13: Order added to Order History
